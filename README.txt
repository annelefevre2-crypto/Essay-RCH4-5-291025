RCH4-AI-OCR-WhiteBlue
========================
Package complet pour GitHub Pages.

- Prise photo ou import (image/PDF)
- OCR sur zone blanche (texte manuscrit/imprimé)
- Lecture du QR dans zone bleue (#0097B2)
- Compilation OCR + QR dans un prompt RCH4 pour ChatGPT

Déploiement :
1. Crée un dépôt public sur GitHub (ex : RCH4-AI-OCR).
2. Dépose tous les fichiers de ce ZIP à la racine.
3. Active GitHub Pages : Settings → Pages → Source: main/(root).
4. Ouvre l’URL fournie (en HTTPS).
